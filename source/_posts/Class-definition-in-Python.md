**Python 3.x:**
```python
class NewClass(object): 
```
new-style class
```python
class NewClass:
```
new-style class implicitly inherits from object

**Python 2.x:**
```python
class NewClass(object):
```
new-style class
```python
class NewClass:
```
old-style class(quite old, don't use)
