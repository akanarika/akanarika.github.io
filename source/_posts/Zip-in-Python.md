Say I have a list:

```python
a = ['.87654321', '2........', '3........', '4........', '5........', '6........', '7........', '8........', '9........']
```

How can I "transpose" it?

```python
b = zip(*a)
```

then 

```python
>>> b
[('.', '2', '3', '4', '5', '6', '7', '8', '9'), ('8', '.', '.', '.', '.', '.', '.', '.', '.'), ('7', '.', '.', '.', '.', '.', '.', '.', '.'), ('6', '.', '.', '.', '.', '.', '.', '.', '.'), ('5', '.', '.', '.', '.', '.', '.', '.', '.'), ('4', '.', '.', '.', '.', ' ', '.', '.', '.'), ('3', '.', '.', '.', '.', ' ', '.', '.', '.'), ('2', '.', '.', '.', '.', ' ', '.', '.', '.'), ('1', '.', '.', '.', '.', ' ', '.', '.', '.')]
```
